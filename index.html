OOP in JavaScript Assignment

Complete the following tasks to demonstrate your understanding of Object-Oriented Programming in JavaScript. Write your JavaScript code in a separate file called assignment.js.

  1. Create a constructor function called Animal that has properties for name and species. Add a method makeSound that logs a sound specific to the animal.
  2. Create two instances of the Animal class: one called dog and another called cat, with appropriate values for their properties.
  3. Create a class called Person using ES6 syntax that has properties for name and age. Add a method greet that logs a greeting message using the person's name.
  4. Create an instance of the Person class called student and call the greet method.
  5. Extend the Person class to create a Student class that adds a property for grade. Override the greet method to include the student's grade in the greeting message.
  6. Create an instance of the Student class and call the overridden greet method.
